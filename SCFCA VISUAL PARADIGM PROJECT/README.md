SCFCA VISUAL PARADIGM PROJECT/  Visual Paradigm project and modeling source files

In addition to the exported diagrams used in the thesis, the repository also preserves the Visual Paradigm project file used to produce and maintain the model-driven 
design artifacts. This supports traceability between the original modeling environment and the final diagrams included in the thesis.
